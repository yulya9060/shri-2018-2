# Задание на вёрстку

Необходимо сверстать панель управления «умным домом». Она показывает информацию о текущем состоянии дома и позволяет управлять устройствами вручную и по сценариям. Дизайнер подготовил для вас отдельные макеты для большого экрана и мобильных телефонов.

# Запуск приложения

- в режиме development
```
    npm -i
    npm run start
```

- в режиме prodaction
```
    npm -i
    npm run build
```
